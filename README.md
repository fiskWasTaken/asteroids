# Asteroids

A simple C++/SFML Asteroids clone.

Features:

* Joystick and keyboard support
* Automatic controller detection
* SAT (Separating Axis Theorem) collision engine
* Rocks of different shapes and sizes
* Infinite waves of rocks
* Multiplayer
* Particles
